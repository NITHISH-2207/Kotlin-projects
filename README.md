# Kotlin Projects

## Aim
To implement basic Kotlin programs using classes, objects, functions, named and default arguments.

---

## Experiment 1: Student Mark Analysis
This program analyzes student marks and displays:
- Total marks
- Average
- Result (Pass / Fail)

Concepts used:
- Class and Object
- Functions
- Named arguments
- Default arguments

---

## Experiment 2: Simple Calculator
A menu-driven calculator where the user selects the operation.

Operations supported:
- Addition
- Subtraction
- Multiplication
- Division

- Kotlin
- IntelliJ IDEA
